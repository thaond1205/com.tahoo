- Đăng thông tin bán căn hộ

    - PROJECT
      - Id
      - Tên dự bán

      - Ban công views
      - Tên tòa nhà
      - Số tầng
      - Mô tả căn hộ
      - Tổng diện tích nhà
      - Chiều dài nhà
      - Chiều rộng nhà
      - Số phòng ngủ
      - Số phòng tắm
      - Diện tích phòng khách 
      - Trạng thái sử dụng
      - Ảnh căn hộ [refer]
      - Thông tin pháp lý [refer]
      - Thông tin khoản vay [refer]
      - Chủ sở hữu [refer]
      - Nội thất [refer]
    - LOAN_INFOS
      - Id 
      - Số tiền vay
      - Trạng thái
    - GALLERIES
      - Id
      - Media_Type
      - Url
    - LEGAL_INFOS
      - Id
      - Trạng thái
    - OWNERS
      - Id
      - Họ Tên
      - Số điện thoại

      - Giới tính
    - FURNITURUES
      - Id
      - Tên tài sản
      - Số lượng
